### Technologies
Project is created with:
* Dart version 2.15.1
* Flutter version 2.8.1
* Bloc State Management
* Rest API

## Setup
To run this project, install it locally using Flutter:

```
$ cd ../movies
$ dart install
$ flutter install
$ flutter start
```
